Overview

Using the correct artwork when creating MeasureOne-branded communications helps to safeguard the consistency and integrity of the visual system. The Downloads section has been designed to facilitate the distribution of logos for appropriate use.

Artwork File Formats

It’s important to choose the correct type of file for your particular application. For best results, we have provided several options for download:

EPS—Preferred, scalable, vector-based format for use with printed materials such as brochures and folders, or any material output to a laser printer. CoreLogic EPS files are Adobe® Illustrator® CS3 EPS. Colors are indicated as spot colors, available as Pantone, CMYK, and RGB breakdowns.

PDF—Scalable, vector-based format. Both CMYK (printed) and RGB (screen) versions are available.

BMP—(Windows bitmap) pixel-based format for use in Microsoft® PowerPoint® or other on-screen applications.

PNG—RGB pixel-based format with transparent background for use in Microsoft PowerPoint or other on-screen applications.

JPG—RGB Pixel-based format for use in Microsoft PowerPoint or other on-screen applications.
